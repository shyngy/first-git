Подключение git в проект

> git init Установит git в проект. Сам файл невидимый, можно к нему перейти: cd .git > git status Статус гита:
branch - ветвление проекта
commit - фиксация проекта
tracked - untracked - отслеживание файлов
gitignore - игнорирует файлы

Основы git

> git add [file]Добавить слежку за файлом > git add . Добавить все файлы и отслеживать > git rm --cached [file] Убрать слежку за файлом > git commit -m "messages" Точка для возврата в проект. "-m" - оставляет комментарии .gitignore создав такой файл в проекте можно игнорировать файлы документы и тд.

branch - ветки

>git branchвывод ветки >git branch [name]создание новой ветки >git checkout -b [name]создание и переход в ветку >git branch -D [name]удаление ветки >git checkout [master]переключение на другую ветвь >git merge [name]брать данные с других веток
